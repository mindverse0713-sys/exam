# Сорил оруулах заавар

## Өгөгдөл бэлэн болсон

Бүх асуулт, хариултууд `scripts/exams-output.json` файлд бэлэн байна.

## Арга 1: Админ интерфейс ашиглах (Зөвлөмж)

1. Вэб сайтад нэвтрэх: `/admin/exams`
2. Нууц үгээ оруулах
3. Анги болон хувилбар сонгох (10-A, 10-B, 11-A, 11-B, 12-A, 12-B)
4. "Шинэ сорил үүсгэх" товч дарах
5. Асуултуудыг оруулах:
   - `scripts/exams-output.json` файлыг нээх
   - Тухайн анги/хувилбарын `sections_public.mcq` массивыг олох
   - Асуулт бүрийг копи хийж оруулах
6. Хариултуудыг оруулах:
   - `answer_key.mcqKey` объектыг олох
   - Асуулт бүрийн зөв хариултыг сонгох
7. "Хадгалах" товч дарах

## Арга 2: API ашиглах (Node.js скрипт)

```bash
# Environment variable тохируулах
export ADMIN_PASS=your_admin_password
export API_URL=http://localhost:3000  # эсвэл production URL

# Скрипт ажиллуулах
node scripts/insert-exams.js
```

**Анхаар:** Энэ арга нь сервер ажиллаж байх шаардлагатай.

## Арга 3: SQL ашиглах (Supabase)

`scripts/exams-output.json` файлыг ашиглан SQL INSERT statement үүсгэх:

```bash
node scripts/generate-sql.js > scripts/insert-exams.sql
```

Дараа нь Supabase SQL Editor дээр `scripts/insert-exams.sql` файлыг ажиллуулах.

## Variant B дарааллын дүрэм

B хувилбар нь A хувилбарын асуултуудыг дараах байдлаар дахин байршуулсан:
- B[1-10] = A[11-20]
- B[11-20] = A[1-10]

Хариултууд мөн ижил дүрмийн дагуу дахин байршуулагдсан.
